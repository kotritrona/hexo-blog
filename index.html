前几天的服务器升级

server, web

大概是3到4号吧，因为服务器实在太久没更新了，就干脆重装了一遍
https://kotri.lv/f/

服务器是挺早以前的了，当时因为第一次装服务器，参考的教程选的CentOS 6.8
结果因为是老系统了，经常碰到兼容问题，比如nginx硬是装不上去，只能用apache
然后apache和websocket不兼容（因为是老版本）又弄了好几天……

这次吸取教训直接装了最新的Ubuntu20.4
就感觉装的特别顺，nginx node php redis ssl几乎都是一次成功

唯一算是有点问题的就是装nodeBB的时候，因为论坛版本也升级了1.12到1.14
里面有几个插件都不兼容，有的需要单独升级，有的只能我自己改代码（还好改起来简单）
倒是我自己写的那个考试插件，一点问题都没有……

还有就是因为服务器用了cloudflare，DNS花了半天
可能是缓存问题吧，不过在cf里面清缓存浏览器里清缓存好像都不管用
虽然换个浏览器就能用了……还是不知道为什么

顺便把数字跳转功能也整合在一起了
比如这个lawn floor的链接是kotri.lv/221
原本的主页（/dd）也去掉换成直接指向论坛了……反正主页也没什么东西w

お前のスパチャで世界を救え

games, galgame, vtuber

挺好玩的一个vtuber相关小游戏
剧本是一个vtuber在8月31日之后消失了，主角回到过去用金钱(SC)拯救Vtuber的故事

虽然本身游戏很短，自己都没实际去玩，看几个vtuber直播就全结局播完了
不过这么短的游戏竟然人物和系统都做得特别漂亮
里面很多剧情似乎都根据SC选项有变化

不过另一个角度想想，像里面的BPex，碟子大叔之类的游戏
可能Vtuber基本都玩过了吧
看过这个游戏就会想，难道“依据SC的数量游戏过程的不同”才是受Vtuber以及其受众喜欢的原因？

然后是那个最后的V-End确实精彩
连人们喜闻乐见的部分都cover到了（バ美肉）
“久 等 了”大概是这种感觉

一个小游戏都能做得这么漂亮
给人一种“啊~不愧是现在风口浪尖的Vtuber行业”的感觉

最后嘛，说到穿越，就想起来9月底holo出那档子事
想想当时，如果有人穿越回一星期以前，能写出什么样的剧本呢？

以前去的很多地方都没人了呢

mdzz

以前的人早就走了，一个个开始上班之后就再也不发帖

想想如果他们现在还在至少打打游戏的话，至少也会稍微发点帖子找点存在感
如果现实中有什么值得说一说的东西，至少也会发点帖子讨论一下

大概大家都一样，工作以后就只有工作了
游戏什么的早就不玩了，网友什么的早就信不过了
再也不可能像以前一样了

生活的目的只剩下挣钱，别的什么都不剩
其他的时间里也只剩下职场应酬和竞争，没有以前那种非营利人际关系

几十年以后变成他们讨厌的大人
养孩子、攀比、给孩子安排工作，重复又一次的循环

其实，从代表这一代人的网络集体意识开始支持“应试教育”的同时
一切的命运就早已注定

世界游戏大全51

games

这就是Switch时代的棋牌游戏大厅吗

里面的AI似乎也是面向新手的，至少看某个Vtuber直播的时候里面象棋和跳棋的AI一直送
当然选的是最低级的AI，最高级的大概就不会了

有些新奇的游戏没听说过，倒是挺有意思的
比如那个ヨッド和花札，花牌还挺好看的，里面还有张牌写着任天堂ww
那个六个不同颜色的蘑菇以前玩过同类型的猜数字，0-9里面猜
好像后来变成组合数学一个经典题目了

这样的游戏实质还是社交游戏吧，主要还是因为有联机功能
不然真要玩单机的话还不如分别下51个.swf游戏w
要不然就是看直播，反正我其实也没买wwwww
